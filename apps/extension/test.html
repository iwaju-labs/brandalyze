Extension Test

This page is for testing the Brandalyze browser extension.

Instructions:

  1. Load the extension in Chrome (chrome://extensions/)
  2. Enable Developer mode
  3. Click "Load unpacked" and select the extension folder
  4. Navigate to Twitter/X and test the functionality

Test Extension Popup:

Click the extension icon in the toolbar to test the popup authentication.

Backend Status:

Backend should be running on: http://localhost:8000/api/

Frontend should be running on: http://localhost:3000/
